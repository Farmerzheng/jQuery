Methods

Method Description/Use
hide Call this to begin the hide animation. 
  // notification is called
  var notification = ...;

  // something happens

  // fade out notification and destroy on complete
  notification.hide();
update Used to update the contents of a notification on the fly. Accepts text and icon in formats as table above denotes. Accepts one or both. 
  // notification is called
  var notification = ...;

  // something happens

  // notification is updated
  notification.update({
    text: "Success!",
    icon: "img/smiley.png"
  });

Browser Support

  • IE7+ — Rounded corners/animation do degrade when not supported, see optional dependencies
  • Google Chrome
  • Firefox
  • Opera
  • Desktop Safari
  • Mobile Safari — iPhone & iPad

Optional Dependencies

  • Spin.js — Required if you want to use a spinner object.
  • jQuery — Required for a fallback CSS animation support.
